使用 CMD

使用 CMD 查看端口占用情况并结束占用的程序

查看所有的端口占用情况
netstat -an0
查看指定端口的占用情况
netstat -an0|findstr "4000"

查看PID对应的进程
tasklist|findstr "1234"
结束进程
taskkill /f /t /im ***.exe
