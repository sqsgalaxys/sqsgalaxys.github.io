buffer 的使用:
下一个 buffer
:bn
上一个 buffer
:bp
打开名称为 filename 的文件
:e<filename>

进制操作:
// 正常打开文件，默认添加一个换行行
vim filename
// 以二进制打开文件
vim -b filename
// 进入 16 进制查看编辑模式,如果在这个模式下执行 :wq 命令会把文件直接保存成这个界面(会把当前界面都保存成字符)
:%xxd
// 修改 16 进制内容后 回到普通模式，可以看到修改产生的变化，这时执行 :wq 不会保存到 16 进制的模式
:%xxd -r
